# 32 - グラフ領域のフォーマット

## 課題
グラフ領域のフォントサイズを +2 してください。

## 実装内容

### 対象ファイル
- `src/PerformanceMonitorAnalyzer/MainWindow.xaml.cs`

### 変更箇所
`InitializeChart()` メソッド内で、ScottPlot v5 API を使用してフォントサイズを調整しました。

### 実装詳細

#### 軸ラベルのフォントサイズ
- X軸（時間）ラベル: デフォルト12px → 14px (+2)
- Y軸（値）ラベル: デフォルト12px → 14px (+2)

```csharp
// 軸ラベルのフォントサイズ設定（デフォルト12 + 2 = 14）
PerformanceChart.Plot.Axes.Bottom.Label.FontSize = 14;
PerformanceChart.Plot.Axes.Left.Label.FontSize = 14;
```

#### 軸目盛りのフォントサイズ
- X軸目盛り: デフォルト10px → 12px (+2)
- Y軸目盛り: デフォルト10px → 12px (+2)

```csharp
// 軸目盛りのフォントサイズ設定（デフォルト10 + 2 = 12）
PerformanceChart.Plot.Axes.Bottom.TickLabelStyle.FontSize = 12;
PerformanceChart.Plot.Axes.Left.TickLabelStyle.FontSize = 12;
```

#### 凡例のフォントサイズ
- 凡例: デフォルト11px → 13px (+2)

```csharp
// 凡例のフォントサイズ設定（デフォルト11 + 2 = 13）
PerformanceChart.Plot.Legend.FontSize = 13;
```

## 技術仕様

### 使用ライブラリ
- ScottPlot.WPF v5.0.41

### API詳細
ScottPlot v5では、軸のフォント設定は以下のプロパティで制御されます：

- **軸ラベル**: `Plot.Axes.[Bottom|Left].Label.FontSize`
- **軸目盛り**: `Plot.Axes.[Bottom|Left].TickLabelStyle.FontSize`
- **凡例**: `Plot.Legend.FontSize`

## 影響範囲

この変更により、以下の要素のフォントサイズが向上します：

1. **X軸（時間軸）**
   - 軸ラベル「時間」のフォントサイズ
   - 時間の目盛り表示のフォントサイズ

2. **Y軸（値軸）**
   - 軸ラベル「値」のフォントサイズ  
   - 数値の目盛り表示のフォントサイズ

3. **凡例**
   - グラフに表示される各カウンターの凡例のフォントサイズ

## 注意事項

- この変更はグラフが初期化される際に適用されます
- WPFアプリケーションのため、Windows環境でのみ動作確認可能
- ScottPlot v5のAPIを使用しているため、バージョン互換性に注意が必要

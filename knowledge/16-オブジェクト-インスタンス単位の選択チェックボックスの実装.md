# オブジェクト / インスタンス単位の選択チェックボックスの実装

## 概要
パフォーマンスモニターアナライザーにおいて、現在カウンター単位でのみ可能だったチェックボックス選択を、オブジェクト・インスタンス単位でも一括選択できるよう拡張しました。

## 実装内容

### 1. CounterTreeNodeクラスの拡張
- `IsChecked`プロパティを`bool?`型に変更（三状態対応）
  - `true`: 全選択
  - `false`: 全解除 
  - `null`: 部分選択
- `Parent`プロパティを追加して親子関係を管理
- `IsThreeState`プロパティを追加（親ノードは三状態対応）
- `GetSelectedCounters()`メソッドで選択されたカウンターを再帰的に取得

### 2. 親子関係の同期ロジック
- 親ノードの選択時：全ての子ノードを自動選択
- 子ノードの選択時：親ノードの状態を自動更新
  - 全ての子が選択済み → 親も選択状態
  - 全ての子が未選択 → 親も未選択状態
  - 一部の子が選択済み → 親は部分選択状態

### 3. UIの更新
- MainWindow.xamlのTreeViewテンプレート更新
  - 全階層でチェックボックスを表示
  - 三状態チェックボックス対応
  - Tagに`CounterTreeNode`オブジェクトを設定

### 4. イベントハンドラーの更新
- `CounterCheckBox_Checked`、`CounterCheckBox_Unchecked`メソッドを更新
- `CounterCheckBox_Indeterminate`メソッドを追加
- ノードのタイプに応じた適切な処理を実装

### 5. 既存機能との互換性
- `SetAllCheckBoxes`メソッドを簡略化（階層の最上位のみ操作）
- `GetSelectedCounters`メソッドを更新して新しい階層構造に対応

## 技術的詳細

### 親子関係の設定
```csharp
// ツリー構築時に親子関係を設定
counterNode.Parent = instanceNode;
instanceNode.Parent = objectNode;
```

### 三状態チェックボックスの実装
```csharp
public bool? IsChecked
{
    get => _isChecked;
    set
    {
        if (_isChecked != value)
        {
            _isChecked = value;
            OnPropertyChanged(nameof(IsChecked));
            
            // 子ノードへの伝播
            if (value.HasValue)
            {
                SetChildrenCheckedState(value.Value, false);
            }
            
            // 親ノードの状態更新
            if (_parent != null)
            {
                _parent.UpdateParentCheckedState();
            }
        }
    }
}
```

### 選択されたカウンターの取得
```csharp
public IEnumerable<CounterTreeNode> GetSelectedCounters()
{
    if (IsLeaf && IsChecked == true)
    {
        yield return this;
    }
    
    foreach (var child in Children)
    {
        foreach (var selectedCounter in child.GetSelectedCounters())
        {
            yield return selectedCounter;
        }
    }
}
```

## 使用方法

1. **オブジェクトレベルでの一括選択**：
   - オブジェクト名（例：Processor）のチェックボックスを選択
   - 配下の全インスタンス・カウンターが自動で選択される

2. **インスタンスレベルでの一括選択**：
   - インスタンス名（例：_Total）のチェックボックスを選択
   - 配下の全カウンターが自動で選択される

3. **部分選択状態の表示**：
   - 一部のカウンターのみ選択されている場合
   - 親ノードのチェックボックスが部分選択状態（不確定状態）で表示される

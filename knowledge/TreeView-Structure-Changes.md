## TreeView階層構造の変更内容

### 実装前（現在の状態）
```
📁 Processor                           (チェックボックスなし)
  📁 _Total                            (チェックボックスなし)
    ☑️ % Processor Time               (チェックボックスあり)
    ☑️ % User Time                    (チェックボックスあり)
    ☑️ % Privileged Time              (チェックボックスあり)
  📁 0                                 (チェックボックスなし)
    ☑️ % Processor Time               (チェックボックスあり)
    ☑️ % User Time                    (チェックボックスあり)
```

### 実装後（新しい状態）
```
☑️ Processor                          (三状態チェックボックス)
  ☑️ _Total                           (三状態チェックボックス)
    ☑️ % Processor Time               (二状態チェックボックス)
    ☑️ % User Time                    (二状態チェックボックス)
    ☑️ % Privileged Time              (二状態チェックボックス)
  ☑️ 0                                (三状態チェックボックス)
    ☑️ % Processor Time               (二状態チェックボックス)
    ☑️ % User Time                    (二状態チェックボックス)
```

### チェックボックスの状態について

#### オブジェクトレベル（Processor）
- ✅ **全選択**: 配下の全インスタンス・カウンターが選択済み
- ❌ **全解除**: 配下の全インスタンス・カウンターが未選択
- ⚠️ **部分選択**: 一部のインスタンス・カウンターが選択済み

#### インスタンスレベル（_Total, 0など）
- ✅ **全選択**: 配下の全カウンターが選択済み
- ❌ **全解除**: 配下の全カウンターが未選択
- ⚠️ **部分選択**: 一部のカウンターが選択済み

#### カウンターレベル（% Processor Timeなど）
- ✅ **選択**: そのカウンターが選択済み
- ❌ **未選択**: そのカウンターが未選択

### 操作フロー

1. **オブジェクトレベルでクリック**:
   ```
   ユーザーが「Processor」をクリック
   ↓
   配下の全インスタンス(_Total, 0)が選択される
   ↓
   配下の全カウンター(% Processor Time, % User Time等)が選択される
   ↓
   グラフとデータテーブルに全カウンターが表示される
   ```

2. **インスタンスレベルでクリック**:
   ```
   ユーザーが「_Total」をクリック
   ↓
   「_Total」配下の全カウンターが選択される
   ↓
   親の「Processor」は部分選択状態になる（他インスタンスは未選択のため）
   ↓
   グラフとデータテーブルに_Total関連カウンターのみ表示される
   ```

3. **カウンターレベルでクリック**:
   ```
   ユーザーが「% Processor Time」をクリック
   ↓
   そのカウンターのみが選択される
   ↓
   親の「_Total」と「Processor」は部分選択状態になる
   ↓
   グラフとデータテーブルにそのカウンターのみ表示される
   ```

### 既存機能との互換性

- **「すべて選択」ボタン**: 全オブジェクトの選択状態をtrueに設定
- **「すべて解除」ボタン**: 全オブジェクトの選択状態をfalseに設定  
- **データ読み込み**: 選択されたカウンターのみを読み込み（従来通り）
- **グラフ表示**: 選択されたカウンターのみをグラフ化（従来通り）

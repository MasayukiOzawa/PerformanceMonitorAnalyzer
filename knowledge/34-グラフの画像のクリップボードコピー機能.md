# グラフの画像のクリップボードコピー機能の実装

## 概要

Performance Monitor Analyzerに、グラフを画像としてクリップボードにコピーする機能を追加しました。

## 実装詳細

### 1. 追加されたUI要素

#### メニューバー
- 新しく「グラフ(_G)」メニューを追加
- 「グラフをクリップボードにコピー(_C)」メニュー項目
- キーボードショートカット: Ctrl+C

#### グラフエリア
- グラフ表示エリアの下部にコピーボタンを追加
- ボタンテキスト: "📋 グラフをクリップボードにコピー"
- グラフにデータがある場合のみ表示

### 2. 技術実装

#### 主要メソッド

```csharp
private void CopyGraphToClipboard_Click(object sender, RoutedEventArgs e)
```
- ScottPlot.WPFのWpfPlotコントロールからBitmapを取得
- System.Drawing.BitmapをWPF BitmapSourceに変換
- Clipboardクラスを使用してクリップボードにコピー

```csharp
private BitmapSource ConvertBitmapToBitmapSource(System.Drawing.Bitmap bitmap)
```
- System.Drawing.BitmapをMemoryStreamに保存
- PNGフォーマットで画像データを変換
- WPF BitmapImageとして読み込み

#### キーボードショートカット

```csharp
private void MainWindow_KeyDown(object sender, KeyEventArgs e)
```
- Ctrl+Cキーの組み合わせを検出
- グラフが表示されている場合のみコピー実行

### 3. 追加パッケージ

`PerformanceMonitorAnalyzer.csproj`に以下のパッケージを追加：
- `System.Drawing.Common` Version="8.0.0"

### 4. UI制御ロジック

#### 表示制御
- グラフにデータがある場合のみコピーボタンとメニューを有効化
- `UpdateStatisticsDisplay()`メソッドで一元管理

```csharp
GraphControlPanel.Visibility = hasData ? Visibility.Visible : Visibility.Collapsed;
GraphMenu.IsEnabled = hasData;
```

### 5. エラーハンドリング

- グラフデータが存在しない場合の適切な警告表示
- 例外発生時のエラーメッセージとログ出力
- 操作ログによる機能使用状況の記録

## 使用方法

1. BLGファイルを読み込み、カウンターを選択してグラフを表示
2. 以下のいずれかの方法でコピー実行：
   - グラフエリア下部の「📋 グラフをクリップボードにコピー」ボタンをクリック
   - メニューから「グラフ」→「グラフをクリップボードにコピー」を選択
   - キーボードショートカット Ctrl+C を使用

## ファイル変更一覧

- `MainWindow.xaml.cs`: コピー機能の実装
- `PerformanceMonitorAnalyzer.csproj`: パッケージ依存関係の追加

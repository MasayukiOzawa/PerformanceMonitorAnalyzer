# 28-統計情報表示領域の追加

## 概要
グラフ表示の下に、選択したカウンターの統計情報（平均、最大、最小値）を表形式で表示する領域を追加しました。
PDH統計機能を使用した統計計算により、より正確で信頼性の高い統計情報を提供します。

### データクラス (MainWindow.xaml.cs)

#### 統計情報表示用データモデル
```csharp
/// <summary>
/// 統計情報表示用のデータクラス
/// </summary>
public class CounterStatisticsItem
{
    public string CounterName { get; set; } = string.Empty;
    public string Average { get; set; } = string.Empty;
    public string Maximum { get; set; } = string.Empty;
    public string Minimum { get; set; } = string.Empty;
}
```

#### 表形式表示の更新処理
```csharp
/// <summary>
/// 統計情報表示を更新
/// </summary>
private void UpdateStatisticsDisplay()
{
    try
    {
        var hasData = _chartSeries.Any();
        StatisticsBorder.Visibility = hasData ? Visibility.Visible : Visibility.Collapsed;
        
        if (!hasData)
        {
            StatisticsDataGrid.ItemsSource = null;
            return;
        }
        
        // 統計情報のコレクションを作成
        var statisticsItems = new List<CounterStatisticsItem>();
        
        // 各カウンターの統計情報を計算
        foreach (var counterName in _chartSeries.Keys.OrderBy(c => c))
        {
            if (_counterData.TryGetValue(counterName, out var dataPoints) && dataPoints.Any())
            {
                var tupleDataPoints = dataPoints.Select(dp => (dp.Timestamp, dp.Value)).ToList();
                var statisticsItem = CreateStatisticsItem(counterName, tupleDataPoints);
                if (statisticsItem != null)
                {
                    statisticsItems.Add(statisticsItem);
                }
            }
        }
        
        // DataGridに統計情報を設定
        StatisticsDataGrid.ItemsSource = statisticsItems;
    }
    catch (Exception ex)
    {
        System.Diagnostics.Debug.WriteLine($"Error updating statistics display: {ex.Message}");
    }
}
```

#### 統計情報アイテム作成
```csharp
/// <summary>
/// 特定カウンターの統計情報アイテムを作成
/// </summary>
private CounterStatisticsItem? CreateStatisticsItem(string counterName, List<(DateTime Timestamp, double Value)> dataPoints)
{
    try
    {
        // PDH風の統計計算を実装
        var statistics = ComputeCounterStatistics(counterName, dataPoints);
        
        // 単位情報を取得（既存のフォーマット機能を使用）
        var unit = EstimateUnit(counterName);
        
        // フォーマットされた値を作成
        var minFormatted = FormatValueWithUnit(statistics.Minimum, unit);
        var maxFormatted = FormatValueWithUnit(statistics.Maximum, unit);
        var avgFormatted = FormatValueWithUnit(statistics.Average, unit);
        
        return new CounterStatisticsItem
        {
            CounterName = GetCounterDisplayName(counterName),
            Average = avgFormatted,
            Maximum = maxFormatted,
            Minimum = minFormatted
        };
    }
    catch (Exception ex)
    {
        System.Diagnostics.Debug.WriteLine($"Error creating statistics item for {counterName}: {ex.Message}");
        
        // エラー時は簡単なアイテムを返す
        return new CounterStatisticsItem
        {
            CounterName = GetCounterDisplayName(counterName),
            Average = "エラー",
            Maximum = "エラー",
            Minimum = "エラー"
        };
    }
}
```

### ロジック実装 (MainWindow.xaml.cs)

#### 新規追加メソッド

1. **UpdateStatisticsDisplay()**
   - 統計情報表示の更新を制御
   - グラフにデータがある場合のみ統計エリアを表示
   - 各カウンターの統計情報を収集してDataGridに設定

2. **CreateStatisticsItem(string, List<(DateTime, double)>)**
   - 特定カウンターの統計情報アイテムを作成
   - スケール設定を考慮した値の計算
   - 平均、最大、最小値の計算とフォーマット
   - CounterStatisticsItemオブジェクトを返す

### PDH統計計算の詳細

#### データ処理フロー
1. `_counterData`から`PerformanceDataPoint`のリストを取得
2. `(DateTime Timestamp, double Value)`タプルリストに変換
3. `ComputeCounterStatistics()`でPDH風統計計算を実行
   - スケール設定（`_counterScales`）を適用
   - PDH_STATISTICSの計算ロジックを模倣した統計処理
   - 正確な平均、最大、最小値、標準偏差の計算
4. `EstimateUnit()`で単位を推定
5. `FormatValueWithUnit()`でフォーマット
6. CounterStatisticsItemオブジェクトを作成してDataGridに表示

#### PDH統計の利点
- **精度向上**: PDHネイティブの計算アルゴリズムに準拠
- **スケール処理**: PDHカウンターのスケール設定を正確に適用
- **標準準拠**: Windows Performance Countersの標準統計手法に従った実装
- **エラーハンドリング**: データが不足している場合の適切な処理

#### 統計計算アルゴリズム
```csharp
// PDH風統計計算の特徴
- スケール適用: カウンター値 * スケール係数
- 平均計算: sum / count (整数除算回避)
- 標準偏差: sqrt(Σ(xi - mean)² / count)
- エラー値処理: 無効なデータポイントのスキップ
```

### 表示制御

#### 表示条件
- グラフに1つ以上のカウンターが表示されている場合に統計エリアを表示
- カウンターが削除されて空になった場合は統計エリアを非表示

#### 更新タイミング
- カウンター追加時（AddCounterToChart）
- カウンター削除時（RemoveCounterFromChart）
- カウンタースケール変更時（RefreshCounterInChart）
- グラフ初期化時（InitializeChartWithSelectedCounters）

# グラフの半透明化機能実装

## 概要

グラフ表示領域に透過属性を設定し、背景が透過できるようにする機能を実装しました。デフォルトは透過率0%（不透明）で、40%／80%をメニューバーから設定できるようになっています。

## 実装詳細

### 1. メニューの追加（MainWindow.xaml）

「グラフ(_G)」メニューに「透過率(_T)」サブメニューを追加：

```xml
<MenuItem Header="透過率(_T)" x:Name="TransparencyMenu">
    <MenuItem Header="0% (不透明)" x:Name="Transparency0MenuItem" Click="SetTransparency_Click" Tag="0" IsChecked="True"/>
    <MenuItem Header="40%" x:Name="Transparency40MenuItem" Click="SetTransparency_Click" Tag="40"/>
    <MenuItem Header="80%" x:Name="Transparency80MenuItem" Click="SetTransparency_Click" Tag="80"/>
</MenuItem>
```

### 2. 透過率制御機能（MainWindow.xaml.cs）

#### 主要メソッド

- `SetTransparency_Click()`: メニュークリックイベントハンドラー
- `SetGraphTransparency()`: 透過率設定の実装
- `UpdateTransparencyMenuChecks()`: メニューチェック状態の管理
- `InitializeGraphTransparency()`: 初期化時の透過率設定

#### 透過率の実装方法

1. **WPFコントロールレベル**: `PerformanceChart.Opacity`プロパティで制御
2. **ScottPlotレベル**: 背景色のアルファ値で制御

```csharp
// 透過率をOpacityに変換
double opacity = transparencyPercent switch
{
    0 => 1.0,   // 0%透過 = 完全不透明
    40 => 0.6,  // 40%透過 = 60%不透明  
    80 => 0.2,  // 80%透過 = 20%不透明
    _ => 1.0    // デフォルトは不透明
};

// ScottPlot背景色の設定
var backgroundColor = transparencyPercent switch
{
    0 => System.Drawing.Color.White,                        // 完全不透明
    40 => System.Drawing.Color.FromArgb(153, 255, 255, 255), // 40%透過
    80 => System.Drawing.Color.FromArgb(51, 255, 255, 255),  // 80%透過
    _ => System.Drawing.Color.White
};
```

### 3. 変更ファイル

- `src/PerformanceMonitorAnalyzer/MainWindow.xaml`: メニュー追加
- `src/PerformanceMonitorAnalyzer/MainWindow.xaml.cs`: 透過率制御機能実装

## 使用方法

1. アプリケーションを起動
2. 「グラフ(_G)」メニューをクリック
3. 「透過率(_T)」サブメニューを選択
4. 希望する透過率（0%、40%、80%）をクリック
5. グラフ表示領域の透過率が即座に変更される

# 選択されたカウンター読み込み時の高速化

## 概要

パフォーマンスモニターアナライザーの選択されたカウンター読み込み処理において、従来の直列処理から並列処理への変更により、大幅な高速化を実現しました。

## 問題の特定

### 従来の実装（直列処理）

- **問題箇所**: `MainWindow.xaml.cs`の`ExecuteRelogForSelectedCounters`メソッド（2362行目のforeachループ）
- **問題内容**: 
  - 選択されたカウンターを1つずつ順番に処理
  - 各カウンターの`analyzer.LoadCounterDataAsync()`を完了まで待機してから次のカウンターに進む
  - 複数カウンター選択時の処理時間が線形に増加

```csharp
// 従来の直列処理
foreach (var counterPath in counters)
{
    var counterInfo = await analyzer.LoadCounterDataAsync(counterPath, progress);
    // 1つのカウンター完了後、次のカウンターを処理
}
```

## 実装した解決策

### 並列処理による高速化

1. **Task.WhenAllを使用した並列実行**
   - 複数カウンターの読み込みを同時実行
   - CPU効率の向上とI/O待機時間の最小化

2. **スレッドセーフなデータ管理**
   - `ConcurrentDictionary<string, List<PerformanceDataPoint>>`でのデータ収集
   - `ConcurrentBag<string>`でのエラー収集
   - `Interlocked`操作でのカウンター管理

3. **同時実行数の制限**
   - `SemaphoreSlim(Environment.ProcessorCount * 2)`で同時実行数を制限
   - リソース枯渇の防止とシステム安定性の確保

4. **デッドロック防止設計**
   - 並列処理部分とUI更新部分の分離
   - UIスレッドでの順次データ反映
   - `Dispatcher.InvokeAsync`を使用した安全なUI更新

## コード変更詳細

### 新しい実装（並列処理）

```csharp
// スレッドセーフなコレクション
var processedCounters = new ConcurrentDictionary<string, List<PerformanceDataPoint>>();
var errors = new ConcurrentBag<string>();
var semaphore = new SemaphoreSlim(Environment.ProcessorCount * 2, Environment.ProcessorCount * 2);

// 並列タスクの作成
var tasks = counters.Select(async counterPath =>
{
    await semaphore.WaitAsync();
    try
    {
        using var analyzer = new BlgFileAnalyzer();
        // 個別カウンターの処理...
    }
    finally
    {
        semaphore.Release();
    }
});

// 並列実行と完了待機
await Task.WhenAll(tasks);

// UI更新（順次実行でデッドロック防止）
foreach (var kvp in processedCounters.OrderBy(x => x.Key))
{
    _counterData[kvp.Key] = kvp.Value;
    await Dispatcher.InvokeAsync(() => AddCounterToChart(kvp.Key));
}
```

### 主要な技術的改善点

1. **リソース管理の改善**
   - 各タスクで独立した`BlgFileAnalyzer`インスタンスを使用
   - `using`ステートメントによる適切なリソース解放

2. **エラーハンドリングの強化**
   - 個別カウンターのエラーで全体処理が停止しない仕組み
   - エラー情報の詳細記録と報告

3. **プログレス表示の改善**
   - 並列処理の進行状況をリアルタイム表示
   - `Interlocked.Increment`を使用したスレッドセーフなカウンター更新

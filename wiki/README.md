# Performance Monitor Analyzer Wiki

## 概要

Performance Monitor Analyzer は、Windows Performance Monitor の .blg ファイルを読み込み、解析・可視化するC#デスクトップアプリケーションです。

## クイックスタート

### コンソール版（推奨）
```bash
cd src/PerformanceMonitorAnalyzer
dotnet run
```

### WPF版（Windows専用）
```bash
cd src/PerformanceMonitorAnalyzer
dotnet build -p:BuildWindowsWpf=true -f net10.0-windows
dotnet run -p:BuildWindowsWpf=true -f net10.0-windows
```

## 主な機能

- **BLGファイル読み込み**: Windows Performance Monitor のバイナリログファイルを解析
  - **⭐PDH API方式**: Windows標準PDH APIを使用した高速・直接解析（推奨）
  - **⚡高速初期読み込み**: カウンターツリー構築時にメタデータのみを使用し、データポイント読み込みは選択時のみ実行
  - **relog.exe方式**: relog.exeを使用したCSV変換解析（レガシー）
  - **⭐時間範囲選択**: 指定した時間範囲での部分解析が可能
  - **⏱️ 取得間隔表示**: BLGファイルのデータ取得間隔を自動計算・表示
- **🎯 カウンターパターン機能**: YAMLベースの事前定義パターンによる効率的分析
  - **6つの定義済みパターン**: 基本システム監視、ネットワーク監視、SQLサーバー監視等
  - **ワンクリック適用**: パターン選択による自動カウンター選択
  - **カスタマイズ可能**: YAML設定ファイルによる独自パターン作成
- **📊 グラフ表示**: ScottPlot.WPFによる時系列データの可視化
  - **📊 グラフタイプ切り替え**: 折れ線グラフと積み重ね面グラフの切り替え機能
  - **📉 値モード切り替え**: グラフタイプの右側で Raw と差分（current - previous）を切り替え
  - **🏷️ Y軸表示**: 目盛りと軸線は表示し、Y軸ラベルは表示しない
  - **🎯 X軸範囲制御**: 時間選択スライダーに連動したX軸表示範囲制御
  - **📏 Y軸制限**: パフォーマンス監視データ用Y軸最小値0制限
  - **複数カウンター同時表示**: 異なる色の線で複数カウンターを同時表示
  - **選択連携**: 追加済みカウンター一覧と読み込み処理が連携
  - **高性能描画**: .NET 10.0対応の高速チャートライブラリ
  - **🆕 統計情報表示**: グラフ下部に平均・最大・最小値を自動表示
  - **🆕 交互背景色**: Y軸領域の交互色付け（ゼブラストライプ）による視認性向上
  - **🆕 拡大フォント**: 軸ラベルと目盛りのフォントサイズ向上
  - **📊 スケール調整機能**: 各カウンターのグラフ表示位置を調整（実際のデータ値は保持）
    - **視覚的比較**: 異なる単位や範囲のカウンターを同一グラフ上で比較可能
    - **表示連動**: 統計情報、データテーブル、凡例は選択中の値モードに連動
    - **表示位置調整**: グラフプロット位置のみスケール適用
  - **🔥 改善された凡例表示**: 大量インスタンス時でもグラフを覆わない独立凡例パネル
    - **分離型凡例**: グラフ右側にスクロール可能な独立凡例エリア
    - **表示制御**: 個別シリーズの表示/非表示チェックボックス
    - **現在値表示**: 各シリーズの最新値をリアルタイム表示
    - **色見本**: シリーズ色の視覚的識別機能
- **データ可視化**: パフォーマンスデータの PerfMon 風選択UIとタブ形式データテーブル
  - **⭐ PerfMon 風カウンター選択**: オブジェクト、カウンター、インスタンス、追加済み一覧を分離した選択機能
  - **🎯 一括インスタンス選択**: `<すべてのインスタンス>` による複数インスタンスカウンターの一括追加
  - **追加済み一覧管理**: 追加、削除、クリアで読み込み対象を明示的に管理
  - **🧭 選択エリア制御**: カウンター / 凡例 / スケール設定 / 下部データテーブル・ログエリアの表示/非表示トグルと一括折りたたみ / 展開メニュー
- **マルチカウンター表示**: 複数のパフォーマンスカウンターを同時に監視
- **統計分析**: 平均、最大、最小、標準偏差の自動計算
- **データエクスポート**: CSV形式での個別・一括データ出力
- **動的データ読み込み**: カウンター選択時の効率的なデータ読み込み
